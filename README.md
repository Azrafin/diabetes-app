# Diabetes App

A web application for diabetes prediction using a machine learning model with a React frontend and a Flask backend.

## Project Structure

```
diabetes-app/
├── backend/
│   ├── app.py
│   ├── diabetes_model.pkl
│   ├── scaler.pkl
│   └── requirements.txt
└── frontend/
    ├── src/
    ├── package.json
    └── ...
```

## Tech Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Flask
- Machine Learning: Scikit-learn
